# MIS SESIONES

Espacio personal donde el usuario puede gestionar y dar seguimiento a todas las sesiones reservadas dentro de VIVE.

Incluye:

- Próximas sesiones.
- Historial de conversaciones con profesionales.
- Mensajes y archivos compartidos.
- Notas y seguimiento de sesiones anteriores.
- Acceso a videollamadas cuando corresponda.

El objetivo es que el usuario tenga un lugar único donde continuar las relaciones y procesos que construye dentro de la plataforma.

**Filosofía:**

Una vez que encontraste a la persona adecuada, la experiencia deja de tratarse de descubrir profesionales y pasa a tratarse de acompañar tu recorrido.
