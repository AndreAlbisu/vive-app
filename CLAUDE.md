# Contexto de base de datos y sesiones

Antes de tocar cualquier código relacionado a base de datos, bookings o salas: leé `SCHEMA.md`.
Antes de empezar a trabajar: leé la última entrada de `CHANGELOG_SESIONES.md` que no sea tuya.
Al terminar la sesión: actualizá ambos archivos con lo que cambiaste.
